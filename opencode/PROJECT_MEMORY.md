# PROJECT MEMORY — OpenCode Ecosystem

## Mission
Build a professional AI-assisted development platform with OpenCode on MacBook as the primary command center.

## Core Areas

1. **Android Development** — Kotlin, Jetpack Compose, Gradle, ADB, Redmi device testing.
2. **React Native Expo** — Cross-platform mobile apps, TypeScript, iOS-style UI.
3. **Telegram Bots** — Automation, lead generation, affiliate management.
4. **WhatsApp Bots** — Conversational automation, webhooks, backend integration.
5. **SaaS** — Scalable backend services.
6. **Affiliate Automation** — Marketing and tracking tools.
7. **Backend (InsForge)** — Auth, Database, Storage, API layer.
8. **Railway / VPS** — 24/7 runtime for bots and workers.
9. **AI Agents** — Specialized agents for each domain.
10. **MCPs** — Filesystem, Terminal, Fetch/Web, Playwright, Dev Tools.
11. **Cloud Memory** — Persistent project memory across sessions.
12. **Metrics & Logs** — Observability and validation.
13. **Safe Recovery** — Backups, snapshots, rollback.
14. **Modular Software** — Clean architecture, scalable design.

## Work Rules

- Think like an engineering team.
- Small, validated changes. Never break the current project.
- Backups before every config modification.
- Prefer free, open-source, stable solutions.
- Ask before dangerous actions.
- Document decisions and architecture.
