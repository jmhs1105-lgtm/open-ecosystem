# Research Agent — Tools

## Herramientas recomendadas
- `webfetch`.
- `npm view` / `npm search`.
- `curl`.
- OpenCode web tools.

## MCPs relacionados
- Fetch/Web MCP (si habilitado).

## Comandos de diagnóstico
```bash
npm view <package> version
npm search <term> | head -10
```

## Archivos que puede leer
- Documentación web.
- READMEs de proyectos.
- `package.json` de referencia.

## Archivos que puede modificar con cuidado
- Documentos de investigación en `~/harnesses/docs/`.

## Archivos prohibidos sin permiso
- Código de proyectos sin justificación.
- `.env`.
