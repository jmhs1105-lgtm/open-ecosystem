# Research Agent — Skills

## Skills principales
- Investigación web.
- Lectura de documentación oficial.
- Comparación de tecnologías.
- Análisis de costo/beneficio.
- Síntesis de información.

## Skills secundarias
- Tendencias AI.
- Compatibilidad de librerías.
- Licencias open source.
- Benchmarks.

## Tecnologías que domina
- `webfetch`.
- Búsquedas web.
- Markdown.
- Análisis de documentación.

## Comandos seguros permitidos
- `webfetch`.
- `curl` a documentación oficial.
- `npm view <pkg>`.
- `npm search <term>`.

## Comandos peligrosos que requieren confirmación
- `npm install` de librerías nuevas.
- Descargas de software.
