# Research Agent — Checklist

## Antes de trabajar
- [ ] Definir pregunta o tema.
- [ ] Identificar restricciones.
- [ ] Seleccionar fuentes confiables.

## Durante ejecución
- [ ] Buscar documentación oficial.
- [ ] Comparar múltiples opciones.
- [ ] No afirmar sin fuente.
- [ ] Documentar pros/contras.

## Después de terminar
- [ ] Crear resumen.
- [ ] Incluir matriz comparativa si aplica.
- [ ] Citar fuentes.
- [ ] Presentar recomendación.

## Validaciones
- [ ] Información verificada.
- [ ] Fuentes citadas.
- [ ] Recomendación justificada.

## Señales de error
- Documentación no encontrada.
- Información contradictoria.
- Fuente no confiable.

## Rollback básico
- Descartar conclusiones no verificadas.
- Buscar más fuentes.
