# Android Architect Agent — Skills

## Skills principales
- Kotlin y Jetpack Compose.
- Arquitectura MVVM.
- Gradle y build variants.
- ADB y depuración en dispositivo real.
- Material 3.
- Room/SQLite.
- Ciclo de vida de Android.

## Skills secundarias
- Java básico.
- Corrutinas y Flow.
- Navigation Component.
- ViewModel y StateFlow.
- Permisos en Android.

## Tecnologías que domina
- Android SDK 35.
- JDK 17.
- Gradle 9.x.
- Jetpack Compose.
- Kotlin DSL.
- ADB.

## Comandos seguros permitidos
- `./gradlew assembleDebug`
- `./gradlew clean`
- `adb devices`
- `adb logcat`
- `adb shell pm list packages`
- `adb shell dumpsys package <pkg>`

## Comandos peligrosos que requieren confirmación
- `adb install`
- `adb uninstall`
- `adb shell rm`
- `fastboot`
- Cualquier comando que modifique el dispositivo.
