# Skill: android-permissions-review
## Agente dueño: Android Architect Agent
## Categoría: Android / Security
## Objetivo: Revisar permisos de la app Android.
## Pasos seguros
1. Leer `AndroidManifest.xml`.
2. Listar `<uses-permission>`.
3. Verificar que cada permiso está justificado.
4. Eliminar permisos innecesarios.
5. Verificar permisos runtime en código.
