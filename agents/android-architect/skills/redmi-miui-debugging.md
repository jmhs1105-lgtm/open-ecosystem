# Skill: redmi-miui-debugging
## Agente dueño: Android Architect Agent
## Categoría: Android / Debug
## Objetivo: Depurar en dispositivos Redmi con MIUI.
## Problemas comunes en MIUI
1. Depuración USB desactivada por defecto.
2. "Instalar vía USB" desactivado.
3. "MIUI optimization" interfiere.
4. Permisos de overlay denegados.
## Pasos seguros
1. Activar Opciones de desarrollador.
2. Activar "Depuración USB".
3. Activar "Instalar vía USB".
4. Desactivar "MIUI optimization" si hay problemas.
5. `adb devices` para verificar.
