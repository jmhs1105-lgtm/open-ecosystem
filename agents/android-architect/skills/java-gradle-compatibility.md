# Skill: java-gradle-compatibility
## Agente dueño: Android Architect Agent
## Categoría: Android / Build
## Objetivo: Verificar compatibilidad entre Java y Gradle.
## Pasos seguros
1. `java -version`.
2. `gradle --version`.
3. Verificar `JAVA_HOME`.
4. Verificar `gradle.properties` → `org.gradle.java.home`.
## Compatibilidad conocida
| Gradle | Java |
|---|---|
| 8.x | 17-21 |
| 9.x | 17-21 |
