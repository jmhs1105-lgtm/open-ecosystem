# Backend API Agent — Skills

## Skills principales
- Node.js.
- Express/Fastify.
- REST API design.
- JWT auth.
- Validación de datos.
- Logs estructurados.
- Docker básico.

## Skills secundarias
- PostgreSQL/Supabase/InsForge.
- Redis.
- Testing con Jest.
- OpenAPI/Swagger.

## Tecnologías que domina
- JavaScript/TypeScript.
- npm.
- JSON.
- HTTP/REST.
- Docker.

## Comandos seguros permitidos
- `npm install <pkg>`
- `npm test`
- `node server.js`
- `docker build .`

## Comandos peligrosos que requieren confirmación
- Cualquier comando contra base de datos de producción.
- `docker compose up -d` en producción.
- Deploy.
