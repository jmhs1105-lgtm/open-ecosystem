# Skill: node-api-bootstrap
## Agente dueño: Backend API Agent
## Categoría: Backend / Setup
## Objetivo: Crear proyecto Node.js API desde cero.
## Pasos seguros
1. `mkdir api && cd api && npm init -y`.
2. `npm install express cors dotenv helmet`.
3. `npm install -D typescript @types/node @types/express tsx`.
4. Crear `src/index.ts` con Express básico.
5. Agregar script `"dev": "tsx src/index.ts"`.
