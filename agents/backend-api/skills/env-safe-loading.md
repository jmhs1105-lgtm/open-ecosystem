## Agente dueño: Backend API Agent
## Categoría: Backend / Security
## Objetivo: Cargar variables de entorno de forma segura.
## Pasos seguros
1. Usar `dotenv` al inicio de la app.
2. Nunca imprimir variables de entorno.
3. Verificar que las variables requeridas existen.
4. Usar valores por defecto solo para no-sensibles.
