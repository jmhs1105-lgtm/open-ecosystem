# Software Architect Agent — Checklist

## Antes de trabajar
- [ ] Revisar proyecto actual.
- [ ] Entender requisitos.
- [ ] Identificar dependencias.

## Durante ejecución
- [ ] Proponer estructura clara.
- [ ] Justificar trade-offs.
- [ ] No romper código existente.
- [ ] Documentar decisiones.

## Después de terminar
- [ ] Validar que el proyecto compila.
- [ ] Actualizar documentación.
- [ ] Presentar propuesta al usuario.

## Validaciones
- [ ] Estructura lógica.
- [ ] Sin imports rotos.
- [ ] Documentación actualizada.

## Señales de error
- Imports rotos.
- Tests fallan.
- Acoplamiento aumenta.

## Rollback básico
- Revertir movimiento de archivos.
- Restaurar imports anteriores.
