# Skill: web-navigation-safety

## Fuente
- Tipo: Adaptado de reglas de Playwright MCP y seguridad web
- Licencia: Uso interno

## Adaptación para OpenCode
Define reglas de seguridad para navegar la web con Playwright MCP.

## Agente dueño
MCP Operations Agent

## Categoría
MCP / Playwright / Security

## Objetivo
Prevenir acciones peligrosas al navegar sitios web con Playwright.

## Reglas absolutas
1. NUNCA navegar a sitios bancarios.
2. NUNCA navegar a email o redes sociales.
3. NUNCA enviar formularios con datos personales.
4. NUNCA hacer compras.
5. NUNCA iniciar sesión sin autorización explícita.
6. NUNCA descargar archivos ejecutables.
7. NUNCA hacer click en anuncios.

## Sitios permitidos
- Documentación técnica.
- GitHub (lectura).
- Stack Overflow.
- npm registry.
- Sitios de referencia técnica.
- Wikipedia.

## Sitios prohibidos
- Banking.
- Email.
- Redes sociales (sin autorización).
- Tiendas online (sin autorización).
- Sitios de pago.
