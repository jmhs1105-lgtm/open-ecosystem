# React Native Expo Agent — Skills

## Skills principales
- React Native y Expo.
- TypeScript.
- UI estilo iOS.
- Navegación con Expo Router.
- Formularios y validación.
- Pantallas de autenticación.
- Conexión backend.

## Skills secundarias
- Gestión de estado (Zustand, Context).
- React Hooks.
- Estilos con StyleSheet y NativeWind.
- Manejo de teclado y formularios.

## Tecnologías que domina
- Expo 56.x.
- React Native 0.74+.
- TypeScript 5.x.
- Node.js.

## Comandos seguros permitidos
- `npx expo start`
- `npx expo doctor`
- `npm install <pkg>`
- `npx create-expo-app`

## Comandos peligrosos que requieren confirmación
- `npx eas build`
- `npx expo publish`
- `npx expo prebuild --clean`
- Instalación global de paquetes.
