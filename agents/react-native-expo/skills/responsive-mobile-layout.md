# Skill: responsive-mobile-layout
## Agente dueño: React Native Expo Agent
## Categoría: Expo / UI
## Objetivo: Crear layouts responsivos que funcionen en diferentes tamaños de pantalla.
## Pasos seguros
1. Usar `Dimensions` para detectar tamaño de pantalla.
2. Usar flexbox para layouts adaptables.
3. Usar `useWindowDimensions()` para cambios dinámicos.
4. Probar en diferentes tamaños.
