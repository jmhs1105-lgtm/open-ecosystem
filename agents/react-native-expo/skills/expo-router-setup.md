# Skill: expo-router-setup
## Agente dueño: React Native Expo Agent
## Categoría: Expo / Navigation
## Objetivo: Configurar Expo Router para navegación file-based.
## Pasos seguros
1. `npx expo install expo-router`.
2. Configurar `app.json` con `scheme`.
3. Crear carpeta `app/` con archivos `_layout.tsx` y `index.tsx`.
4. Usar `Stack` o `Tabs` para navegación.
