# Skill: auth-form-validation
## Agente dueño: React Native Expo Agent
## Categoría: Expo / Validation
## Objetivo: Validar formularios de autenticación.
## Validaciones
- Email: formato válido, no vacío.
- Password: mínimo 8 caracteres, al menos 1 número.
- Confirm password: debe coincidir.
- Nombre: no vacío, mínimo 2 caracteres.
