# Skill: insforge-mobile-auth
## Agente dueño: React Native Expo Agent
## Categoría: Expo / InsForge
## Objetivo: Conectar autenticación de InsForge desde la app móvil.
## Pasos seguros
1. Configurar `.env` con `INSFORGE_URL` e `INSFORGE_API_KEY`.
2. Crear servicio de auth que llame a InsForge.
3. Implementar login, registro, logout.
4. Guardar token de sesión.
5. Manejar refresh token.
