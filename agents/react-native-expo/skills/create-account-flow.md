# Skill: create-account-flow
## Agente dueño: React Native Expo Agent
## Categoría: Expo / Auth
## Objetivo: Crear flujo de registro de usuario.
## Pasos seguros
1. Crear pantalla de registro con campos: email, password, confirm password.
2. Validar inputs (email formato, password mínimo).
3. Llamar a API de registro.
4. Mostrar loading durante request.
5. Manejar errores (email ya existe, password débil).
6. Redirigir a pantalla principal tras éxito.
