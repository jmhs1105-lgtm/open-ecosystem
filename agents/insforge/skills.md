# InsForge Agent — Skills

## Skills principales
- Diseño de bases de datos SQL.
- Row Level Security (RLS).
- Esquemas de autenticación.
- Storage policies.
- InsForge SDK/CLI.

## Skills secundarias
- PostgreSQL.
- Migraciones.
- API REST.
- Auth JWT.

## Tecnologías que domina
- InsForge CLI (`npx @insforge/cli`).
- SQL.
- `.env` management.
- OpenCode MCP de InsForge.

## Comandos seguros permitidos
- `npx @insforge/cli link --project-id <id>`
- `npx @insforge/cli db list`
- Lectura de esquemas.

## Comandos peligrosos que requieren confirmación
- Cualquier `DROP`.
- Cualquier `DELETE`.
- Cualquier modificación de RLS en producción.
