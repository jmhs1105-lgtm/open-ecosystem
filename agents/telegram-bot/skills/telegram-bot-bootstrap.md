# Skill: telegram-bot-bootstrap
## Agente dueño: Telegram Bot Agent
## Categoría: Telegram / Setup
## Objetivo: Crear bot de Telegram desde cero.
## Pasos seguros
1. `npm install telegraf`.
2. Crear bot con `new Telegraf(process.env.BOT_TOKEN)`.
3. Registrar comando /start.
4. Agregar middleware de logging.
5. Ejecutar con polling o webhook.
