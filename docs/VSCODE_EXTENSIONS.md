## Installation

VS Code will prompt to install recommended extensions when opening the workspace.
Or: `Cmd+Shift+P` → "Extensions: Show Recommended Extensions"

## Per-Language

### JavaScript / TypeScript
- ESLint + Prettier (required)

### React Native / Expo
- React Native Tools (recommended)
- ES7+ React snippets (optional)

### Python
- Python (if needed for scripts)

### SQL
- SQLite Viewer (for .sql files)

### Docker
- Docker (for Railway/VPS configs)

### Markdown
- Markdown All in One (optional)

## Rules

- No API keys in settings.json
- No tokens in any .vscode/ file
- Extensions are recommendations, not requirements
- Don't install extensions automatically without asking
