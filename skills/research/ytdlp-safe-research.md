# Skill: yt-dlp Safe Research

## Objective

Use yt-dlp safely and legally for research, education, and analysis. Strict policy: only own content, educational material, or authorized downloads.

## When to Use

- Downloading your own YouTube/Twitch/etc content
- Educational material research
- Audio extraction for transcription (own content)
- Technical analysis of public content

## When NOT to Use

- Pirating content
- Bypassing DRM
- Downloading private/paid content
- Evading subscriptions

## Agent Responsible

**Research Agent**

## Commands

```bash
# Install
brew install yt-dlp

# Audio extract
yt-dlp -x --audio-format mp3 URL

# Subtitle extract
yt-dlp --write-subs URL
```

## Rules

1. Document source and purpose before download
2. Never used for piracy
3. Never used for DRM bypass
4. Never used for private content
5. Delete after research use

## Validation

- [ ] Source documented
- [ ] Purpose documented
- [ ] Content is public/own
- [ ] No terms of service violation
